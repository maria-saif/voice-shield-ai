# Voice Shield – AI Voice Phishing Detection System

Voice Shield is an AI-based system designed to detect voice phishing (vishing) attacks by analyzing voice recordings and transcribed text.

## Project Overview
The system converts speech to text using Speech-to-Text technology and analyzes the content using Natural Language Processing (NLP) and machine learning techniques to detect suspicious patterns such as impersonation attempts, urgency expressions, and financial manipulation.

## Features
- Voice recording analysis
- Speech-to-Text processing
- Natural Language Processing (NLP)
- Fraud pattern detection
- Risk scoring system

## Technologies Used
- Python
- Flask
- Scikit-learn
- Speech-to-Text
- NLP

## Purpose
The goal of this project is to enhance cybersecurity and help detect voice phishing attacks that target users through fraudulent phone calls.
